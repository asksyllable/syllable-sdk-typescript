---
title: 'Event'
---

An event represents a specific occurrence during a session.

## Example Usage

```typescript
import { Event } from "syllable-sdk/models/components";

let value: Event = {
  timestamp: new Date("2024-06-18T21:49:56.662Z"),
  source: "tool-call",
  category: "automation",
  type: "phone-transfer",
  userId: "+18042221111",
  description: "Transfer to phone number +18042221111",
  attributes: {},
};
```
