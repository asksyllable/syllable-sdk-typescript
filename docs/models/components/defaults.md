# Defaults

The default values for the parameters of the function/tool call.


## Supported Types

### `components.One`

```typescript
const value: components.One = {};
```

### `{ [k: string]: components.ToolParameterDefault }`

```typescript
const value: { [k: string]: components.ToolParameterDefault } = {
  "key": {
    transform: {},
  },
};
```
