# ToolHttpEndpoint

The configuration for an HTTP API call by a tool.

## Example Usage

```typescript
import { ToolHttpEndpoint } from "syllable-sdk/models/components";

let value: ToolHttpEndpoint = {
  url: "https://celebrated-skeleton.info/",
  method: "post",
  argumentLocation: "path",
};
```
