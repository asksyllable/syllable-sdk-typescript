# PromptLlmConfig

LLM configuration for a prompt.

## Example Usage

```typescript
import { PromptLlmConfig } from "syllable-sdk/models/components";

let value: PromptLlmConfig = {
  version: "2024-05-13",
  apiVersion: "2024-06-01",
  temperature: 1,
  seed: 123,
};
```
